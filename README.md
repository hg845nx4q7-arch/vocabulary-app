# 儿童背单词 - KET/PET 词汇学习应用

一个帮助儿童学习剑桥英语 KET 和 PET 词汇的互动式网页应用。

## 功能特点

- **双词库支持**：包含 KET 和 PET 两个级别的词汇
- **学习模式**：通过卡片式学习，显示单词、音标、翻译和例句
- **测试模式**：拼写测试，逐字母输入验证
- **错题本**：自动记录错误次数≥2的单词，支持专项复习
- **进度保存**：自动保存学习进度到本地，关闭后再打开不丢失
- **游戏化设计**：金币奖励系统，连击加分机制
- **语音发音**：支持单词发音功能（需浏览器支持 Web Speech API）

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- LocalStorage 数据持久化
- Web Speech API 语音合成
- 响应式设计，支持移动端

## 学习数据

- **KET 词库**：约 600 个基础词汇
- **PET 词库**：进阶词汇
